# Open in Diffshub

Chrome extension. On a GitHub PR page, click the toolbar icon (or press `⌘⇧D` / `Ctrl+Shift+D`) to open the same PR in [diffshub.com](https://diffshub.com).

See: https://x.com/pierrecomputer/status/2057174934674124941?s=20

## Install

`chrome://extensions` → Developer mode → **Load unpacked** → pick this folder.
