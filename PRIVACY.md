# Privacy Policy

**Open in Diffshub** does not collect, store, transmit, or share any personal data.

## What the extension accesses

- When you click the toolbar icon on a GitHub pull request page, the extension reads the URL of the active tab to extract the owner, repository, and pull request number.
- That information is used only to construct the equivalent `https://diffshub.com/<owner>/<repo>/pull/<number>` URL and open it in a new tab.

## What the extension does not do

- Does not read, modify, or inject scripts into any page.
- Does not track browsing activity.
- Does not collect analytics or telemetry.
- Does not send any data to the extension author or any third party.
- Does not use cookies or local storage.

## Permissions

- `activeTab` — used only on click to read the current tab's URL.
- `declarativeContent` — used to enable the toolbar icon only on GitHub pull request URLs.

## Contact

For questions, open an issue at the project's repository.
